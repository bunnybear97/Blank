# Space Portfolio (Next.js)

A second, space-themed portfolio for Subhasree Babu, adapted from
[Jenin82/SpacePortfolio](https://github.com/Jenin82/SpacePortfolio).

This is a separate Next.js + React Three Fiber project and is kept alongside
the main static site (`index.html` at the repo root). It is not deployed to
GitHub Pages automatically since it requires a Node build.

## Run locally

```bash
cd space-portfolio
npm install
npm run dev
```

## Deploy

Deploy this folder to a platform like Vercel (set the project root to
`space-portfolio`).
