# 3D Portfolio (Next.js)

A third portfolio variant for Subhasree Babu, adapted from
[Naresh-Khatri/3d-portfolio](https://github.com/Naresh-Khatri/3d-portfolio).

This is a separate Next.js project and is kept alongside the main static
site (`index.html` at the repo root) and `space-portfolio/`. It requires a
Node build, so it is not deployed via GitHub Pages.

## Run locally

```bash
cd 3d-portfolio
npm install
npm run dev
```

## Deploy

Deploy this folder to a platform like Vercel (set the project root to
`3d-portfolio`).
